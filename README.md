# 智能页面代码生成工具

基于 **组件库 + 模板 + AI 推断** 的智能代码生成工具，通过极简交互快速生成高质量的 Vue2 业务页面代码。

## ✨ 核心特性

- 🎯 **极简交互**：只需选择组件，无需配置属性
- 🤖 **AI 智能推断**：自动推断组件配置、API 配置、依赖关系
- 📦 **组件库驱动**：支持基础组件和自定义业务组件
- 🎨 **模板系统**：基于 Handlebars 模板，代码质量稳定
- ⚡ **快速生成**：3-5 分钟生成完整页面代码

## 🎯 产品定位

**基于组件库和模板的智能代码生成工具**

- 用户只负责"选择"（模板和组件）
- AI 负责"推断和生成"（配置和代码）
- 模板保证代码质量和稳定性

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:5173

### 3. 使用流程

```
Step 1: 选择模板
  → 从模板库选择页面类型（标准列表页/复杂筛选页/表单配置页）
  
Step 2: 选择组件
  → 在 Slot 位置点击"添加组件"
  → 从组件选择器选择需要的组件（输入框/选择器/业务组件）
  
Step 3: 配置 API
  → 填写主数据 API 地址
  → 填写关联 API 地址（系统自动识别）
  
Step 4: 生成代码
  → 点击"生成代码"
  → AI 自动推断配置并生成完整代码
  → 复制或下载代码
```

## 🏗️ 技术架构

### 三层架构

```
页面模板层
  - 标准列表页
  - 复杂筛选页
  - 表单配置页
     ↓ 使用
业务组件层
  - 区域选择器（双面板）
  - 人员选择器（标签输入）
  - 任务表格（动态）
     ↓ 基于
基础组件层
  - Element UI
  - hui-pro
```

### AI 的作用

```
用户选择组件
  ↓
AI 推断配置
  - 组件属性（字段名、标签、校验）
  - API 配置（请求参数、响应格式、数据映射）
  - 组件依赖（区域 → 团队 → 查询）
  ↓
模板生成代码
  - Handlebars 模板（页面结构）
  - AI 生成逻辑（API 调用、数据流转、事件处理）
  ↓
输出 Vue 代码
```

├── src/
