# AI 辅助可视化低代码页面生成工具

基于 Vue3 + Vite 的智能代码生成工具，通过**可视化拖拽 + 配置 + AI 生成**的方式，快速生成高保真的 Vue2 + hui 组件库页面代码。

## ✨ 核心特性

- 🎨 **可视化编辑器**：类低代码平台的三栏布局（组件库 + 实时预览 + 属性配置）
- 🧩 **灵活组装**：模板提供智能布局容器，用户自由拖拽配置组件
- 🔗 **多接口支持**：一个组件可绑定多个接口（选项、提交、联动等）
- 🤖 **AI 增强**：通过明确的配置和提示词，AI 生成连接代码
- 📦 **高保真输出**：生成可直接使用的 Vue2 SFC 代码，包含完整业务逻辑

## 🎯 核心理念

### 职责清晰
- **模板**：提供智能布局容器（h-page-search、h-page-table 等）
- **用户**：选择组件、配置接口、编写提示词（业务决策）
- **AI**：根据配置生成连接代码（data、methods、watch 等）

### 工作流程
```
选择模板 → 拖拽组件 → 配置接口 → AI生成代码
   ↓          ↓          ↓            ↓
  布局骨架   业务组件   数据绑定    完整代码
```

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:5173

### 3. 使用流程

1. **选择模板**：从顶部下拉框选择页面模板（标准列表页/左树右表等）
2. **拖拽组件**：从左侧组件库拖拽组件到预览区的 slot
3. **配置组件**：在右侧配置面板设置属性、绑定接口
4. **生成代码**：点击"生成代码"，AI 自动生成完整的 Vue2 代码
